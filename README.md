## Khair Morocco Donation Website
this website designed to promote unity, donations, and volunteering for students in Morocco. 
Built with HTML, CSS, and basic JavaScript, it serves as a platform for a student-led initiative to support underprivileged children through clothing donations and community outreach. 
The site emphasizes accessibility, mobile-friendliness, and a clean, modern design inspired by Moroccan themes.
## Tools
-HTML5 for structure.
-CSS3 (custom styles + Tailwind CSS).
-Fonts: Google Fonts (e.g., Playwrite NG Modern, Russo One).
-Images: Local assets (e.g., logos, backgrounds).
